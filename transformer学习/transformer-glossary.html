遇到术语时先看“一句话”，再看“在模型里做什么”。

术语一句话在模型里做什么
Token文本切分后的单位模型读取和生成的基本单位，不一定等于一个完整单词。
Embeddingtoken 的数字表示把离散 token ID 映射成可计算的连续向量。
Positional Encodingtoken 的位置标记让 Attention 能区分相同 token 出现在不同位置。
Parameter训练过程中学到的数字包括 Wq、Wk、Wv、FFN 和输出层权重。
Activation本次输入产生的中间结果Q/K/V、隐藏状态等都属于 activation，不是长期固定参数。
Query当前 token 想寻找什么与所有 Key 比较，产生匹配分数。
Key每个 token 的匹配标签告诉 Query 自己是否值得被关注。
Value真正被读取的内容根据 Attention 权重被加权汇总。
Logitsoftmax 前的原始分数可以为任意实数，不是概率。
Softmax把分数变成比例输出非负、总和为 1 的权重或词表概率。
Head一套独立的关注视角多个 head 可以并行寻找不同类型的关系。
Residual把输入直接加回输出保留原信息，并改善深层网络的梯度传播。
LayerNorm整理每个 token 的特征尺度让网络数值更稳定。
FFN逐 token 的小型神经网络对 Attention 汇总后的特征进行非线性加工。
Causal Mask遮住未来位置保证 Decoder 生成当前 token 时不能偷看未来。
Cross-AttentionDecoder 查询 EncoderQ 来自 Decoder，K/V 来自 Encoder。
Forward Pass从输入计算到输出工作簿展示的主要过程。
Backpropagation根据错误反向计算梯度属于训练过程，本工作簿没有展示。
